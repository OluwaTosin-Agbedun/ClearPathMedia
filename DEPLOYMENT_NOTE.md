# ClearPath Media TV Production Deployment Note

## Staging vs. Production Preview

The `share.google` link generated by Google AI Studio serves exclusively as a **temporary, sandboxed staging/preview environment**. It is meant for testing, alignment verification, and iterative feedback.

Deploying to production under the actual custom domain **`https://www.clearpathmediatv.com`** is handled via your custom deployment targets (such as Vercel, Firebase Hosting, or Cloud Run).

### Main Highlights
- **Target URL**: `https://www.clearpathmediatv.com`
- **Fallback Crawlable SEO**: Built directly into static `index.html` to support fast search indexers and social media platforms before client hydration.
- **YouTube API Integration**: Configure the `VITE_YOUTUBE_API_KEY` in your production hosting platform secrets (or enter manually in the Admin control panel) to synchronize the video discovery suite cleanly.
